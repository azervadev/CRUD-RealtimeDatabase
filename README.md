# CRUD Realtime Database

Android sample project focused on CRUD operations with Firebase Realtime Database.

## Overview

This project demonstrates how to create, read, update and delete data in Firebase Realtime Database using Kotlin and an MVVM-based Android architecture.

## Tech Stack

* Kotlin
* Android
* Firebase Realtime Database
* MVVM
* Coroutines
* StateFlow
* Dagger-Hilt
* Clean Architecture

## Features

* Create data in Firebase Realtime Database
* Read data from Firebase
* Update existing records
* Delete records
* State management with StateFlow
* Dependency injection with Dagger-Hilt
* Clean project structure

## Architecture

The project follows an MVVM approach with separation between UI, state management and data access.

## Project Status

Learning/demo project focused on Firebase Realtime Database integration and CRUD operations in Android.
